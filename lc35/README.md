# Leetcode Question 35. Search Insert Position

**Question Description**<br>
Given a sorted array of distinct integers and a target value, return the index if the target is found. If not, return the index where it would be if it were inserted in order.

You must write an algorithm with O(log n) runtime complexity.


**Test cases**<br>
**Input:** nums = [1,3,5,6], target = 5 <br>
**Output:** 2 <br>


**Approach**<br>
This question shows the simplly implementation of binary search, The question askes the index of the target which can be simpelly optained by binary search.<br>

**Time Complexity - O(log (n)) <br>
Space Complexity - O(1)**
